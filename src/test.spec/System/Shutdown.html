Shutting Apollo down

The user interface needs to know when Apollo is starting the shutdown sequence so that it can safely terminate the different controls and forms and their data.

During the shutdown process Apollo first checks that all parts of the system can indeed be stopped. If this is the case then Apollo notifies the user interface indicating that the shutdown process has begon. This process should then run succesfully.
