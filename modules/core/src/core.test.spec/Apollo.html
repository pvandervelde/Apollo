Overview

The following pages provide access to the different specification tests for the Apollo API. These specifications will load the Apollo application and manipulate this application through the same API that the User Interface (UI) uses. The goal is to verify that the integration of the UI business layer with the Apollo framework works as specified.

Apollo core functionality.

  • Loading and starting Apollo.
  • Shutting Apollo down.
  • Gathering information about the current state of Apollo.
