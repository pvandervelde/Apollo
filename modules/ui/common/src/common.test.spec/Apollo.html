Overview

The following pages provide access to the different regression tests for the Apollo API. These tests will load the Apollo application and manipulate this application through the same API that the User Interface (UI) uses. The goal for these tests is to verify that the integration of the UI business layer with the Apollo framework has been done correctly.

Note that these tests are regression tests only, not UI tests. At no point will UI controls be manipulated.

Starting and stopping Apollo.

The first tests will test the behavior of the system for start-up and shutdown.
